# 오픈소스프로그래밍 팀프로젝트 9팀 사용자 설명서(README)

시연영상 유튜브 URL : https://www.youtube.com/watch?v=zEDx4OPzJUI

__1. login.html__
<img width="1792" alt="login" src="https://user-images.githubusercontent.com/72423836/123110282-e5b4e880-d476-11eb-9b18-35b6662ed34f.png"><br>
(1) 이전에 회원가입을 했을 경우 사용자의 아이디와 비밀번호를 입력하여 ‘Sign in’ 버튼을 누른다. (아이디나 비밀번호가 일치하지 않을 경우, 로그인이 되지 않는다.)<br>
(2) 이전에 회원가입을 하지 않아 사이트가 처음일 경우, ‘Register’버튼을 누른다.
<br><br>
__2. register.html__
<img width="1792" alt="register" src="https://user-images.githubusercontent.com/72423836/123110400-054c1100-d477-11eb-9582-42ba75bd6b4a.png"><br>
(1) 사용자가 원하는 아이디와 비밀번호를 입력한 후, ‘submit’버튼을 누른다. (submit 버튼을 누른 후 다시 login.html 창으로 이동한다.)
<br><br>
__3. home.html__
<img width="1792" alt="home" src="https://user-images.githubusercontent.com/72423836/123110445-11d06980-d477-11eb-9dbd-d527b16a6205.png"><br>
(1) 사용자의 위치를 선택할 수 있게 설정한 두 개의 콤보박스에서 대구의 ‘어느’구, ‘어느’동에 살고있는지 현 위치를 선택한 후 우측의 버튼(‘에 살고있습니다!’)을 클릭한다.<br>
(2) ‘로그아웃’ 버튼을 누를 시 로그아웃되고 다시 로그인 페이지로 이동한다.<br>
(3) ‘리뷰 게시판’ 버튼을 누를 시 리뷰를 작성하거나 볼 수 있는 리뷰 게시판으로 이동한다. 
<br><br>
__4. food.html__
<img width="1792" alt="food" src="https://user-images.githubusercontent.com/72423836/123110556-2ca2de00-d477-11eb-8114-5abc98bd95c4.png"><br>
(1) 상단의 로고(‘날름날음’) 클릭 시 홈 페이지(home.html)로 이동한다.<br>
(2) 화면 우측 상단의 ‘로그아웃’버튼 클릭시 로그아웃되며 로그인 화면(login.html)으로 이동한다.<br>
(3) 상단 중앙부에 현 위치의 현재 날씨(rainy, snowy, sunny, cloudy)를 알려주는 아이콘 표시하였고, 아이콘 바로 아래에 현재 위치와 현재 날씨, 온도를 알려준다.<br>
(4) 하단에 추천 음식들과 사용자의 실시간 선택 기록을 통해 추정된 상위 3개의 음식의 순위을 참고한다. <br>
(5) 나열된 음식들 중에서 먹고싶은 음식을 중앙의 콤보박스를 통해 선택하고 바로 우측의 버튼(‘이게 먹고 싶어요!’) 선택한다.
<br><br>
__5. storelist.html__
<img width="1792" alt="storelist" src="https://user-images.githubusercontent.com/72423836/123110609-36c4dc80-d477-11eb-9fb4-8e36ba5c43c5.png"><br>
(1) 상단의 로고(‘날름날음’) 클릭 시 홈 페이지(home.html)로 이동한다.<br>
(2) 박스 안에 사용자가 선택한 음식을 파는 현위치 주변의 음식점 3곳 나열한다.<br>
(3) 리뷰가 궁금하거나 리뷰를 작성하고 싶을 때 리뷰 게시판(음식점 리뷰보기)으로 이동한다.<br>
(4) 다른 음식도 알아보고 싶다면 로고 우측의 콤보박스에서 다른 음식을 선택한 후 ‘이게 먹고 싶어요!’ 버튼을 클릭한다.
<br><br>
__6. reviewboard.html__
<img width="1792" alt="reviewboard" src="https://user-images.githubusercontent.com/72423836/123110651-3f1d1780-d477-11eb-9f1b-a5b54968b9d6.png"><br>
(1) 상단의 로고를 클릭하면 홈 페이지로 이동한다.<br>
(2) 로그아웃 버튼을 누를 시 로그아웃되고 다시 로그인 페이지로 이동한다.<br>
(3) 리뷰를 남기고 싶은 식당과 사용자의 리뷰를 식당과 한줄평 입력창에 입력한 후, 작성하기 버튼을 누른다.<br>
(4) 리뷰를 작성하면, 리뷰 목록에 사용자가 입력한 리뷰가 입력되게 된다. <br>
